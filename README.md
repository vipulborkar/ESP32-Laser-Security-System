# IoT-Based-Laser-Security-System-Real-Time-Intrusion-Alerting-via-ESP32-and-Telegram-Bot
IoT-based Laser Security System using ESP32, LDR, and Telegram Bot. Detects laser beam interruptions in real time, activates local alarms, and sends instant Telegram notifications to multiple users. Features Wi-Fi connectivity, NTP time synchronization, and intrusion monitoring.
